# Legal and Ethical Considerations

This document outlines the legal and ethical boundaries for the Unreal Engine 4 (UE4) Lua scripting pipeline described in this repository.

The purpose of this section is to clearly define **intent**, **scope**, and **responsibility**, and to ensure that the project is used only for legitimate, educational, and research-oriented purposes.

---

## Project Intent

This project is intended to:

- Document Lua scripting pipeline architecture in UE4
- Explain runtime discovery and lifecycle-based execution concepts
- Support educational study, tooling development, and modding research
- Provide a neutral reference for understanding script VM integration

The project does **not** aim to enable misuse, exploitation, or circumvention of protections.

---

## No Targeting of Commercial Games

This repository:

- Does not target any specific commercial game
- Does not include game names, identifiers, or offsets
- Does not distribute proprietary binaries or assets
- Does not provide ready-to-use tooling against live applications

All concepts are presented in a **generic and abstracted** form.

---

## Respect for Software Licenses

Users of this project are responsible for:

- Complying with Unreal Engine licensing terms
- Respecting the licenses of any Lua integrations or plugins
- Following the terms of service of applications they interact with
- Ensuring that any experimentation is permitted by applicable licenses

This repository does not grant permission to violate any software license.

## Educational and Research Use Only

The content of this repository is provided strictly for:

- Learning and understanding engine internals
- Academic or personal research
- Development of legitimate tools or frameworks
- Exploration of scripting systems in controlled environments

It is not intended for cheating, bypassing safeguards, or unauthorized modification of software.

## Ethical Guidelines

When working with engine internals and scripting systems, users should adhere to the following ethical principles:
